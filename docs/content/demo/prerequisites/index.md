## Tested with
* OpenShift v4.7.8
* `oc` v4.7.8
* `vault` v1.5.4
* `helm` v3.4.2


## You will need a user who is able to create:
* `ClusterRole`
* `ClusterRolebinding`
* `MutatingWebhookConfiguration`
* `Deployment`
* `ServiceAccount`
* `ConfigMap`
* `Statefulset`
* `Route`
* `NetworkPolicy`
* `PersistentVolumeClaim`
